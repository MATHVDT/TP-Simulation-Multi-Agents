TP - Architectures Logicielles et Qualité : Simulation Multi-Agents *(SMA)*

Réalistion d'un simulation multi-agents. Deux types d'agents (équipe ROUGE/BLEU) s'affrontent sur une carte composée de cases hexagonales. Le but des agents est de couvrir le plus de cases possibles sur la carte.
