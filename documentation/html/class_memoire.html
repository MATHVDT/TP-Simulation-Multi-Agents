◆ getInfluence()

float Memoire::getInfluence ( int  differenceLevel)

Calcul l'influence de l'agent sur celui-ci.

Paramètres
intdifferenceLevel - Différence de level entre les 2sagents

Calcul l'influence de l'agent qui transmet sa mémoire en fonction de la différence de niveau.

Le taux d'influence suit une fonction affine : y = a * x + b Taux d'influence : Agent transmetteur +9 level / agent receveur => influence = 0.9 Agent transmetteur -9 level / agent receveur => influence 0.1
